Baseball Pitching Types

There are 3 different types of Pitches. Fastball, Breaking balls, and Change ups(off-speed pitch).

  • Fastball:

    1. Four-seam Fastball
    2. Two-seam Fastball
    3. Cutter
    4. Sinker
    5. Split-finger Ball

  • Breaking Balls:

    1. Curveball
    2. Slider
    3. Slurve
    4. 12-6 Curveball
    5. Knuckle Curve
    6. Screwball

  • Chnage-ups(Off-speed pitch):

    1. Change up
    2. Circle Change up
    3. Forkball
    4. Knuckle Ball
